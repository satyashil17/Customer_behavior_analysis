## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed initial inspection of the data.

### 2. Exploratory Data Analysis (EDA)

* Examined data structure and summary statistics.
* Identified trends, distributions, and relationships.
* Created visualizations for better understanding.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data.
* Converted data types where required.

### 4. SQL Analysis

* Imported cleaned data into PostgreSQL, MySQL, and SQL Server.
* Wrote SQL queries to:

  * Filter and sort data
  * Aggregate metrics
  * Perform joins
  * Generate business insights

### 5. Power BI Dashboard

* Built an interactive dashboard.
* Added KPIs and charts.
* Included filters and slicers for dynamic analysis.

### 6. Report Creation

* Summarized findings and insights.
* Documented methodology and analysis results.

### 7. Presentation

* Created a professional presentation using Gamma.
* Highlighted objectives, methodology, dashboard, and key insights.

## How to Run

1. Clone or download this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook or Python script.
4. Load the dataset.
5. Run the data cleaning and EDA steps.
6. Execute the SQL scripts in PostgreSQL, MySQL, or SQL Server.
7. Open the Power BI file (`.pbix`) to explore the dashboard.
8. Review the report and Gamma presentation for project insights.
